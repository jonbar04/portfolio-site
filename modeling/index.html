Computer Animation: Modeling Final Project

For my final project, I decided to model Flint's house from the game MOTHER 3.
This was a rather difficult process, since I had to go and take a setting envisioned from a single perspective and put it in a three dimensional space. This class was my first ever time working with modeling, so I had to learn a lot of skills along the way. But overall, I think I did a pretty alright job with the final result.
